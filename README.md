# SIEDL

## Sistema de Incentivación Educativa y Dignificación Laboral

Propuesta socioeducativa y socioeconómica orientada a incentivar la formación oficial, dignificar trabajos esenciales y estudiar mecanismos de redistribución funcional del excedente económico.
